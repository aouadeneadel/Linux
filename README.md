À propos de ce script

Ce script est un outil d'automatisation tout-en-un conçu pour les développeurs utilisant des systèmes basés sur Debian et Ubuntu, comme Linux Mint. Il configure votre environnement de travail rapidement et de manière cohérente en installant les éléments suivants.

Fonctionnalités et installations
Éditeurs et IDEs
 - VS Code
 - Sublime Text
 - Eclipse

Langages de programmation
 - PHP
 - Java
 - OCaml

Outils de virtualisation

 - Docker
 - VirtualBox

Serveurs et bases de données
 - XAMPP
 - MySQL
 - DBeaver

Applications de productivité
 - Onlyoffice
 - Lecteurs PDF
 - Simple Note

Fonctionnalités supplémentaires
Génère automatiquement des raccourcis de bureau pour les applications installées.
Personnalise le fond d'écran de votre machine.
