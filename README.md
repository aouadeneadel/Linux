# Linux interface :

Cette version, plus détaillée, est idéale pour le haut de votre fichier README.md afin de donner un aperçu complet du projet.
​Ce script est un outil d'automatisation tout-en-un pour les développeurs sur les systèmes basés sur Debian et Ubuntu, comme Linux Mint.
​Il permet de configurer votre environnement de travail rapidement et de manière cohérente en installant :
​    - Des éditeurs de code et IDEs (VS Code, Sublime Text, Eclipse)
​    - Des langages de programmation (PHP, Java, OCaml)
​    - Des outils de virtualisation (Docker, VirtualBox)
​    - Des serveurs locaux et clients de bases de données (XAMPP, MySQL, DBeaver)
​    - Des applications de productivité (LibreOffice Writer, lecteurs PDF)
​Le script génère même automatiquement des raccourcis de bureau pour les applications installées et personnalise le fond d'écran de votre machine.
